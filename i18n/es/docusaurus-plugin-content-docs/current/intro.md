# Introducción al Tutorial

Vamos a descubrir **Docusaurus en menos de 5 minutos**.

## Empezando

Comienza **creando un nuevo sitio**.

O **prueba Docusaurus inmediatamente** con **[docusaurus.new](https://docusaurus.new)**.

### Lo que necesitarás

- [Node.js](https://nodejs.org/en/download/) versión 20.0 o superior.
